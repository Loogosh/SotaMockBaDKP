# Дефолтные настройки аддона SotA

## Основные настройки (Bidding)

| Настройка | Значение | Описание |
|----------|----------|----------|
| `SOTA_CONFIG_AuctionTime` | `20` | Время аукциона в секундах |
| `SOTA_CONFIG_AuctionExtension` | `8` | Продление аукциона в секундах |
| `SOTA_CONFIG_EnableOSBidding` | `1` | Включить MS/OS Ставки (1 = включено, 0 = выключено) |
| `SOTA_CONFIG_EnableZoneCheck` | `1` | Автоначисление DKP по зоне рейда (1 = включено, 0 = выключено) |
| `SOTA_CONFIG_EnableOnlineCheck` | `1` | Проверка онлайна для начисления DKP (1 = включено, 0 = выключено) |
| `SOTA_CONFIG_AllowPlayerPass` | `1` | Разрешить игрокам пропускать ставки (1 = включено, 0 = выключено) |
| `SOTA_CONFIG_DisableDashboard` | `0` | Скрыть Dashboard (0 = показан, 1 = скрыт) |
| `SOTA_CONFIG_OutputChannel` | `WARN_CHANNEL` | Канал вывода сообщений (обычно Raid Warning) |

## Misc DKP (Дополнительные настройки)

| Настройка | Значение | Описание |
|----------|----------|----------|
| `SOTA_CONFIG_UseGuildNotes` | `0` | Хранить DKP в публичных заметках гильдии (0 = офицерские заметки, 1 = публичные заметки) |
| `SOTA_CONFIG_MinimumBidStrategy` | `1` | Стратегия минимальной ставки:<br>0 = Без минимальных правил<br>1 = Минимальное увеличение на 100 ДКП<br>2 = Минимальное увеличение на 10% |
| `SOTA_CONFIG_DKPStringLength` | `5` | Длина строки DKP в заметках гильдии |
| `SOTA_CONFIG_MinimumDKPPenalty` | `50` | Минимальный штраф DKP при процентном срезе |

## Messages (Сообщения)

Все сообщения инициализируются с русскими переводами по умолчанию через функцию `SOTA_VerifyEventMessages()`.

## Прочие настройки

| Настройка | Значение | Описание |
|----------|----------|----------|
| `SOTA_CONFIG_Messages` | `nil` | Таблица сообщений (инициализируется автоматически) |
| `SOTA_CONFIG_VersionNumber` | `nil` | Номер версии конфигурации |
| `SOTA_CONFIG_VersionDate` | `nil` | Дата последнего изменения |
| `SOTA_CONFIG_Modified` | `false` | Флаг изменения конфигурации |
| `SOTA_HISTORY_DKP` | `{ }` | История DKP транзакций |

## Примечания

- Все настройки сохраняются в файл `SavedVariables/SOTA.lua` при выходе из игры
- При первом запуске аддона все настройки устанавливаются в дефолтные значения
- Если настройка имеет значение `0`, она считается выключенной (для boolean настроек)
- Если настройка имеет значение `1`, она считается включенной (для boolean настроек)
- Для сброса всех настроек к дефолтным значениям используйте команду `/sota resetconfig`
