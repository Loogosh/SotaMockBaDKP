# DKP Loot Assistant - Руководство по использованию

## Обзор

DKP Loot Assistant — модуль для SOTA, автоматизирующий процесс разыгрывания лута:
- Автоматическое сканирование эпиков при открытии лута
- Запуск аукционов SOTA одной кнопкой
- Прямая выдача предметов победителям из окна лута или через трейд

---

## Установка

Модуль интегрирован в SOTA. После копирования папки `SotA/` в `Interface/AddOns/`:

1. Перезапустите игру или выполните `/reload`
2. Проверьте загрузку: `/la help`

---

## Использование

### Автоматическое сканирование лута

При открытии лута (как Master Looter) автоматически:
1. Сканируются все предметы
2. Фильтруются по качеству (по умолчанию: эпики)
3. Открывается окно Loot Assistant со списком

### Интерфейс

**Окно Loot Assistant:**
- **Список предметов** с иконкой, названием и статусом:
  - 🟡 "Ожидает разрола" — можно запустить аукцион
  - 🟠 "Аукцион идёт..." — идёт разыгрывание в SOTA
  - 🟢 "Победитель: PlayerName" — можно выдать предмет
  - ⚪ "Выдан: PlayerName" — предмет уже выдан

**Кнопки для каждого предмета:**
- `[Roll]` — запустить аукцион SOTA для этого предмета
- `[Выдать]` — автоматически выдать предмет победителю
- `[X]` — удалить предмет из списка

**Управление:**
- `[Разыграть Всё]` — запустить аукционы для всех предметов по очереди
- `[Очистить]` — очистить весь список
- `[Закрыть]` — закрыть окно (данные сохраняются)

### Команды

```
/la show    — открыть окно
/la hide    — скрыть окно
/la clear   — очистить список лута
/la help    — показать справку
```

## Workflow (Типичный сценарий)

### 1. Открытие лута

```
1. Убили босса
2. Открыли лут (Master Looter)
   → Автоматически открывается Loot Assistant
   → Список эпиков отсканирован
```

### 2. Разыгрывание предметов

**Вариант A: По одному предмету**
```
1. Нажать [Roll] у нужного предмета
   → Запускается аукцион SOTA
2. Игроки делают ставки через MOCKBA-DKP клиент
3. Принять ставку в SOTA UI ([Accept])
   → Победитель автоматически назначается в Loot Assistant
```

**Вариант B: Все предметы сразу**
```
1. Нажать [Разыграть Всё]
   → Аукционы запускаются по очереди с задержкой 2 сек
2. После каждого аукциона принять победителя
```

### 3. Выдача предметов

**Из окна лута (приоритетный способ):**
```
1. Предмет разыгран, победитель назначен
2. Окно лута ещё открыто
3. Нажать [Выдать]
   → Предмет автоматически выдаётся через GiveMasterLoot()
   → Сообщение в рейд-чат: "Выдан [Item] → PlayerName"
```

**Через трейд (если окно лута закрыто):**
```
1. Закрыли лут, предмет попал в сумку
2. Победитель открывает трейд с вами
   → Автоматически появляется Trade Helper
3. Нажать [Выдать Всё]
   → Все предметы для этого игрока помещаются в трейд
```

## Настройки

В будущем будет UI для настроек. Сейчас через lua:

```lua
-- Минимальное качество (3=синие, 4=эпики, 5=легендарки)
/script SOTA_CONFIG_LA_MinQuality = 4

-- Автооткрытие окна при LOOT_OPENED
/script SOTA_CONFIG_LA_AutoOpen = 1  -- 1=да, 0=нет

-- Исключать квестовые предметы
/script SOTA_CONFIG_LA_ExcludeQuest = 1  -- 1=да, 0=нет

-- Применить

```

## Особенности и ограничения

### ✅ Что работает

- Автоматическое сканирование лута при открытии
- Запуск аукционов SOTA из списка
- Автоматическое назначение победителя после аукциона
- Прямая выдача из окна лута через `GiveMasterLoot()`
- Fallback на трейд, если окно лута закрыто
- Обработка очереди аукционов ("Разыграть Всё")
- Уведомление клиентов через addon-канал

### ⚠️ Известные ограничения

1. **GiveMasterLoot работает только пока открыто окно лута**
   - Если закрыли окно — используйте трейд
   - Trade Helper автоматически откроется при трейде с победителем

2. **SlotIndex меняется после выдачи**
   - Модуль автоматически обновляет индексы при `LOOT_SLOT_CLEARED`

3. **Очередь аукционов**
   - SOTA не рассчитана на параллельные аукционы
   - "Разыграть Всё" запускает по одному с задержкой 2 сек

4. **Требуется роль Master Looter**
   - Сканирование и выдача работают только у ML
   - Обычные игроки видят только уведомления через клиент

## Интеграция с MOCKBA-DKP

Loot Assistant отправляет уведомления клиентам:

```
Формат: 1;LOOT_ASSIGN;itemId=19019;winner=PlayerName
Канал: MDKP_SRV (addon)
```

Клиенты с MOCKBA-DKP видят:
- Уведомление о победе
- Предмет в своём UI
- Инструкцию открыть трейд с мастером


```
[Loot Assistant] Отсканировано предметов: 3
[Loot Assistant] Победитель: PlayerName (500 ДКП)
[Loot Assistant] Выдан Thunderfury игроку PlayerName
```

## Troubleshooting

**Окно не открывается:**
```lua
/dump getglobal("SOTA_LootAssistantFrame")  -- должно быть table
/la show
```

**Предметы не сканируются:**
- Проверьте, что вы Master Looter
- Проверьте качество: `/script DEFAULT_CHAT_FRAME:AddMessage(SOTA_CONFIG_LA_MinQuality)`

**Кнопка [Выдать] не работает:**
- Окно лута должно быть открыто
- Игрок должен быть в рейде/группе
- Используйте трейд как fallback

**Аукцион не запускается:**
- Проверьте права: нужен ранг помощника или лидера рейда
- Проверьте состояние SOTA: `/dump SOTA_GetAuctionState()`
