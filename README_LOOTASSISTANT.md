# DKP Loot Assistant - Реализовано

## ✅ Что реализовано

### Основные модули

1. **LootScanner** (`sota-lootassistant.lua`)
   - Автоматическое сканирование лута при `LOOT_OPENED`
   - Фильтрация по качеству (по умолчанию: эпики 4+)
   - Фильтрация квестовых предметов
   - Структура данных: `SOTA_ScannedLoot` с полями itemId, link, normLink, quality, slot, winner, status

2. **SOTAIntegration** (`sota-lootassistant.lua`)
   - Запуск аукционов SOTA через `SOTA_StartAuction()`
   - Хук на `SOTA_AcceptBid()` (дополнительно), но основное назначение победителя идёт через парсинг чата
   - Хук на `SOTA_CancelAuction()` для сброса статуса
   - Отправка уведомлений клиентам через addon-канал (`MDKP_SRV`)

3. **TradeAssistant** (`sota-lootassistant.lua`)
   - **Режим A:** Прямая выдача через `GiveMasterLoot(slot, candidateIndex)`
   - **Режим B:** Fallback на трейд с поиском предметов в сумках
   - Автоматическое определение партнёра в трейде
   - Trade Helper UI для выдачи предметов

4. **LootWindow UI** (`sota-lootassistant.xml`)
   - Главное окно `SOTA_LootAssistantFrame` (450x500px)
   - 9 фиксированных строк `SOTA_LootItem1..9` по шаблону `SOTA_LootItemTemplate`
   - Для каждой строки: иконка, название, статус, кнопки [Roll], [Выдать], [X]
   - Верхняя панель для ручного добавления предмета (поле ввода + кнопка «Добавить»)
   - Нижние кнопки: [Выдать Всё], [Закрыть]

5. **Trade Helper UI** (`sota-lootassistant.xml`)
   - Окно `SOTA_LA_TradeHelperFrame` для выдачи через трейд
   - Список предметов для текущего партнёра
   - Кнопки: [Выдать Всё], [Закрыть]

### События

Обрабатываются:
- `LOOT_OPENED` — сканирование лута, автооткрытие окна
- `LOOT_SLOT_CLEARED` — обновление индексов слотов
- `LOOT_CLOSED` — пометка для fallback на трейд
- `TRADE_SHOW` — определение партнёра, показ Trade Helper
- `TRADE_CLOSED` — скрытие Trade Helper
- `CHAT_MSG_RAID_WARNING` / `CHAT_MSG_RAID` — парсинг сообщений SOTA для назначения победителя

### Команды

```
/la show    — открыть окно
/la hide    — скрыть окно
/la clear   — очистить список лута
/la test    — создать 3 тестовых предмета
/la debug   — диагностика (видимость окна, строк, содержимое SOTA_ScannedLoot)
/la help    — показать справку
```

### SavedVariables

Добавлены в `SOTA.toc`:
- `SOTA_CONFIG_LA_MinQuality` — минимальное качество (по умолчанию: 4)
- `SOTA_CONFIG_LA_AutoOpen` — автооткрытие окна (по умолчанию: 1)
- `SOTA_CONFIG_LA_ExcludeQuest` — исключать квестовые (по умолчанию: 1)
- `SOTA_LootHistory` — история лута (опционально)
- `SOTA_ScannedLoot` — текущий отсканированный лут

### Интеграция в SOTA Core

В `sota-core.lua` добавлено:
- Функция `SOTA_SendLootAssignMessage(itemId, winner)`
- Инициализация конфигов в `SOTA_ResetAllSettingsToDefault()`

## 🔄 Workflow

### 1. Открытие лута
```
Master Looter открывает лут
  ↓
Автоматическое сканирование эпиков
  ↓
Окно Loot Assistant открывается со списком
```

### 2. Разыгрывание
```
Нажать [Разрол] (в строке предмета или любой строке)
  ↓
Запуск аукциона SOTA
  ↓
Игроки делают ставки через MOCKBA-DKP клиент
  ↓
Мастер принимает ставку в SOTA UI
  ↓
Победитель автоматически назначается в Loot Assistant
```

### 3. Выдача
```
Если окно лута открыто:
  Нажать [Выдать] или [Выдать Всё]
    ↓
  GiveMasterLoot() выдаёт напрямую
    ↓
  Статус: "Выдан"

Если окно лута закрыто:
  Победитель открывает трейд
    ↓
  Trade Helper автоматически открывается
    ↓
  Нажать [Выдать Всё]
    ↓
  Предметы из сумок помещаются в трейд
```

## 🐛 Известные ограничения

1. **GiveMasterLoot работает только пока окно лута открыто**
   - Решение: автоматический fallback на трейд

2. **SlotIndex меняется после выдачи предмета**
   - Решение: обработка LOOT_SLOT_CLEARED, автообновление индексов

3. **Очередь аукционов с задержкой 2 сек**
   - SOTA не рассчитана на параллельные аукционы
   - Решение: OnUpdate обработчик с очередью

4. **Требуется роль Master Looter**
   - Это ожидаемое поведение

## 🚀 Быстрый старт

1. Скопируйте папку `SotA/` в `Interface/AddOns/`
2. Перезапустите игру или `/reload`
3. Откройте лут как Master Looter
4. Окно Loot Assistant откроется автоматически
5. Нажмите [Roll] для запуска аукциона
6. После победы нажмите [Выдать]
