# 🎮 Новые кнопки Dashboard - Инструкция

## 📍 Расположение кнопок:

```
Dashboard:
┌───────────────────────────────────────────────┐
│  [👥]  [💀]  [💰]  [📕/⚔️]                   │
│                                               │
│  SOTA Master: ИмяМастера                     │
└───────────────────────────────────────────────┘

[👥] = Raid Queue (очередь в рейд)
[💀] = Raid Boss DKP (за босса с именем)
[💰] = Add Raid DKP (начислить всем фикс. сумму)
[📕] = Officer Notes / [⚔️] = Public Notes
```

---

## 💀 Кнопка 1: "Raid Boss DKP" (череп)

### Назначение:
Начисление DKP **за убийство босса** (с именем босса в истории)

---

## 💰 Кнопка 2: "Add Raid DKP" (монетка)

### Назначение:
Начисление **фиксированной суммы** DKP всему рейду (без имени)

### Что делает:
Каждый игрок получает **указанное количество** DKP

### Как использовать:

**Пример 1: Начислить 100 DKP всем**
```
1. Нажал 💰
2. Окно: "100" (предзаполнено)
3. "Начислить"
→ Каждый в рейде получил +100 DKP ✅
```

**Пример 2: Начислить 50 DKP**
```
1. Нажал 💰
2. Окно: "100"
3. Изменил на: "50"
4. "Начислить"
→ Каждый получил +50 DKP ✅
```

**Эквивалентная команда:**
```
/sota raid +100 DKP
```

### Отличие от 💀:

| Кнопка | Формат | В истории | Использование |
|--------|--------|-----------|---------------|
| 💀 Череп | `100 BossName` | "100 DKP - Ragnaros" | За убийство босса |
| 💰 Монета | `100` | "100 DKP - DKP" | Общее начисление |

---

## 💀 Кнопка 3: "Raid Boss DKP" (череп)

### Что делает:
Быстрое начисление DKP за убийство босса

### Как использовать:

#### ✅ Правильное использование (зона = иконка):

**Пример 1: В MC с Public Notes**
```
Ты в: Molten Core
Иконка: ⚔️ Thunderfury (Public Notes)

1. Убили Ragnaros
2. Нажал 💀
3. Открылось окно: "600 " (предзаполнено)
4. Дописал: "600 Ragnaros"
5. "Начислить"
→ Всему рейду +600 DKP в Public Notes ✅
```

**Пример 2: В Naxx с Officer Notes**
```
Ты в: Naxxramas
Иконка: 📕 Atiesh (Officer Notes)

1. Убили Kel'Thuzad
2. Нажал 💀
3. Открылось окно: "1200 "
4. Дописал: "1200 Kel'Thuzad"
5. "Начислить"
→ Всему рейду +1200 DKP в Officer Notes ✅
```

---

#### ⚠️ Несовпадение зоны и иконки (первый раз):

**Пример 3: В MC, но иконка показывает Officer**
```
Ты в: Molten Core (Public Notes зона)
Иконка: 📕 Atiesh (Officer Notes настроено)

1. Убили Golemagg
2. Нажал 💀
3. ПОЯВЛЯЕТСЯ ПРЕДУПРЕЖДЕНИЕ:
   ┌────────────────────────────────────┐
   │ ВНИМАНИЕ!                          │
   │                                    │
   │ Ты в зоне: MC/Ony/BWL/AQ          │
   │ Но начислишь DKP в: Накс/Кара     │
   │                                    │
   │ Всё правильно?                     │
   ├────────────────────────────────────┤
   │ [Да, начислить]  [Отмена]         │
   └────────────────────────────────────┘

4а. Нажал "Да, начислить"
    → Открывается диалог: "600 "
    → Дописал: "600 Golemagg"
    → "Начислить"
    → DKP начислено в Officer Notes ✅
    → Запомнено подтверждение!

4б. Нажал "Отмена"
    → Ничего не происходит
```

**Пример 4: В MC второй раз (после подтверждения)**
```
Ты в: Molten Core
Иконка: 📕 Atiesh (Officer Notes)

1. Убили Lucifron
2. Нажал 💀
3. Сразу диалог: "600 "  ← БЕЗ предупреждения!
   (подтверждение запомнено)
```

**Сброс подтверждения:**
- При переключении иконки 📕⚔️
- При входе в другую зону

---

#### 🌍 В открытом мире:

**Пример 5: В Orgrimmar**
```
Ты в: Orgrimmar (не рейд)
Иконка: 📕 Atiesh

1. Нажал 💀
2. Сразу диалог: "100 "  ← БЕЗ предупреждения
3. Вводишь: "50 За посещаемость"
4. "Начислить"
→ DKP начислено в Officer Notes ✅
```

### Как переключать:

**Перед рейдом Naxx/Kara:**
```
1. Проверь иконку
2. Должна быть 📕 Atiesh
3. Если ⚔️ Thunderfury → нажми на кнопку
4. Иконка меняется → 📕 Atiesh
5. В чат: "⚙️ Переключено на Officer Notes (Накс/Кара DKP)"
```

**Перед рейдом MC/Ony/BWL/AQ:**
```
1. Проверь иконку
2. Должна быть ⚔️ Thunderfury
3. Если 📕 Atiesh → нажми на кнопку
4. Иконка меняется → ⚔️ Thunderfury
5. В чат: "⚙️ Переключено на Public Notes (MC/Ony/BWL/AQ DKP)"
```
